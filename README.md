# Code Dark Syntax theme

A dark syntax theme inspired on **Visual Studio Code**, with some tweaks.

## Supported Languages

I tested it with a few languages, and intend to add more:

* Pascal
* Java
* Coffeescript
* JavaScript
* Markdown
* Python
* LESS
* JSON
